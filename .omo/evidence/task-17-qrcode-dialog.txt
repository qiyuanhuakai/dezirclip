Task 17: QrCodeDialog + useQrCode hook
=======================================
Date: 2026-06-18

## Deliverables

### 1. src-tauri/src/app/commands/qr_cmd.rs (NEW, ~60 LOC)
- Tauri command `generate_qr_png(content, size_px) -> String` wrapping services::qr
- Converts Vec<u8> PNG bytes to data:image/png;base64 data-URL
- Also exposes `generate_qr_svg(content) -> String` for copy-as-SVG
- 3 inline tests: data_url format, SVG content, empty content

### 2. src/features/clipboard/hooks/useQrCode.ts (NEW, ~60 LOC)
- React hook wrapping generate_qr_png Tauri command
- Returns: { dataUrl, loading, error, generate }
- Cache: single-entry cache by content+size key (cacheRef)
- Stale state protection: generationIdRef counter, discards stale results
- Empty content: returns error without calling invoke

### 3. src/features/clipboard/components/QrCodeDialog.tsx (NEW, ~100 LOC)
- Modal dialog with forwardRef, role="dialog", aria-modal="true"
- Props: entry (ClipboardEntry), onClose
- ESC key closes dialog (document keydown listener)
- Click outside (overlay) closes dialog
- QR image centered, content text truncated at 60 chars
- Copy-as-SVG button: invokes generate_qr_svg, writes to navigator.clipboard
- Chinese placeholder text (no new i18n keys)
